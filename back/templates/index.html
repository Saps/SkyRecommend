Все API находится на http://185.221.152.242:5480/api/ 

Для работы с API необходимо получить Bearer токен. Он получается при авторизации и используется
во всех остальных запросах:
POST /api/user/login
| {
| 	 "username" : "user2",
| 	 "password" : "u2"
| }
Ответ:
{
  "access_token": "e5c7b3c1-d1e9-4adc-992f-3108b54dffb8",
  "user_role": "user"
}
Токен отправляется в заголовке, как показано ниже:
GET /api/user/current
Authorization: Bearer a18ffc42-6159-4da2-ba47-3bbcf4bed922

работа алгоритма по выбору сервисов

GET
/findservs
работа алгоритма по выбору сервисов исходя из истории сделок

GET
/listserv
общий список сервисов, отфильтрованный по параметрам. Параметры:
      type - высвечиваемый тип сервисов
      search - часть символьной строки для поиска
      offset - смещение
      limit - лимит (для пагинации)

GET
/listservtypes
      Типы сервисов (массив строк)

GET
/tunealgor
      Список алгоритмов для настройки

POST
/tunealgor
      Настройка параметров алгоритмов. Пример JSON:
      [
	{
		"id":1,
		"weight": 0.8,
		"is_enabled" : 1
	},
	{
		"id":3,
		"weight": 1.5,
		"is_enabled" : 0
	}
]

GET
/servtext
      Выборка корпуса текстов для определенного сервиса. serv_id - какой именно сервис

GET
/servcond
      Алгоритм поиска релевантных слов в корпусе текстов, serv_id - какой именно сервис

GET
/servgraph
      Алгоритм построения графа семантических ядер для текстов, serv_id - какой именно сервис

POST
/updater
      Ендпойнт для апдейта рейтинга
